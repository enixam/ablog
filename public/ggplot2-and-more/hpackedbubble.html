进阶 R 图形

3 hpackedbubble

hpackedbubble 提供了 highchart.js 中 packed bubble chart 在 R 语言 中的接口，同名的核心函数 hpackedbubble() 主要绘制两种图形：packed bubble charts 和 split packed bubble charts.

library(tidyverse)
library(hpackedbubble)

hpackedbubble 自带的数据集：CO2（2014 年各国碳排放量）:

CO2 %>% as_tibble()

##    continent country          CO2

##  2 Europe    Croatia         20.7
##  3 Europe    Belgium         97.2
##  4 Europe    Czech Republic 112. 
##  5 Europe    Netherlands    158. 
##  6 Europe    Spain          242.

基于以上数据，绘制一个基本的 split packed bubble chart，注意函数的调用方法:

hpackedbubble(cat = CO2$continent, 
              name = CO2$country, 
              value = CO2$CO2)

split = 0 绘制 packed bubble chart:

hpackedbubble(cat = CO2$continent, 
              name = CO2$country, 
              value = CO2$CO2，
              split = 0)

注意，这两种图呈现的实际上都是一维数据。

hpackedbubble() 的重要参数：

  • cat: 分组变量，决定气泡的填充色和 split packed bubble chart 中的嵌套关系
  • name: 气泡的静态标签。为避免重叠，hpackedbubble() 会自动隐藏一部分 value 小标签，通过 dataLabelsFilter控制( 默认 250).

dataLabels = 0 取消静态标签

dataLabelsFormat 用 HTML 标签自定义静态标签的格式, {point.name} 和 {point.y}表示该气泡的 name 和 value

dataLabelsColor 指定静态标签的颜色，默认为黑色

pointFormat： 鼠标悬停时的动态标签，设定与 dataLabelsformat 同理

  • value: 气泡的大小

  • split = 1 split packed bubble chart （嵌套）； split = 0 packed bubble chart

  • theme：配色、背景、字体、变宽等主题设置，可选 darkgreen/darkblue/avocado/darkunica/gray/ gridlight/grid/sandsignika (默认) /sunset

  • witdh 和 height 设置图表的宽高，默认分别为 “100 %”

  • title、titleAlign (“left”、“center”、“right”)、titleSize(ex. “20px”) 和 titleColor(ex. “#333333”) 设置主标题

  • subtitle、subtitleAlign、subtitleSize 和 subtitleColor 设置副标题

  • packedbubbleMinSize 和 packedbubbleMaxSize 控制气泡大小的标度，分别默认 “50%” 和 “250%”

  • gravitional 气泡弹出的力度，默认 0.0625

  • dragBetweenSeries = 1: 绘制 split packed bubble plot 是否可以在分组之间拖拽气泡

  • parentNodeLimit = 1: 气泡是否严格在组内浮动

hpackedbubble(CO2$continent, CO2$country, CO2$CO2,
              theme = "darkgreen",
              title = "CARBON EMISSIONS AROUND THE WORLD (2014)",
              pointFormat = "<b>{point.name}:</b> {point.y}m CO<sub>2</sub>",
              dataLabelsFilter = 100,
              packedbubbleMinSize = "50%",
              packedbubbleMaxSize = "150%",
              split = 1,
              gravitational = 0.02,
              parentNodeLimit = 1,
              titleColor = "white",
              width = "100%")
hpackedbubble(CO2$continent, CO2$country, CO2$CO2,
              theme = "avocado",
              title = "CARBON EMISSIONS AROUND THE WORLD (2014)",
              pointFormat = "<b>{point.name}:</b> {point.y}m CO<sub>2</sub>",
              dataLabelsFilter = 100,
              packedbubbleMinSize = "50%",
              packedbubbleMaxSize = "250%",
              gravitational = 0.5,
              titleColor = "Black",
              width = "120%")
hpackedbubble(CO2$continent, CO2$country, CO2$CO2,
              theme = "darkblue",
              title = "CARBON EMISSIONS AROUND THE WORLD (2014)",
              pointFormat = "<b>{point.name}:</b> {point.y}m CO<sub>2</sub>",
              dataLabelsFilter = 500,
              titleColor = "White",
              height = "600px")
